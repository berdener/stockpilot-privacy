StockPilot – Privacy Policy

Effective Date: 30/11/2025
App Name: StockPilot
Developer: Bekir Erdener

StockPilot does not collect, store, share, or sell any personally identifiable customer data.
The app only reads and updates product, variant, SKU, and inventory quantities through Shopify’s Admin API to provide inventory management functionality.

1. Information We Access

StockPilot only interacts with the following Shopify store data:

Product names

Variant names

SKUs

Inventory quantities

Inventory locations

No customer information, order details, payment information, or personal data is ever accessed.

2. How We Use the Data

StockPilot processes inventory data only to sync stock quantities between Shopify and your app panels.
No collected data is stored externally.

3. Data Sharing

StockPilot does not share any data with third parties.

4. Data Storage

We do not store Shopify data on external servers. All data stays inside Shopify.

5. Uninstallation

When a merchant uninstalls StockPilot, all app permissions are automatically revoked via Shopify.
No data is kept or retained.
